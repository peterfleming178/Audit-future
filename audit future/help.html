HELP CENTER

  • CALCULATOR

    • This button is used to open calculator included with your operating system

  • GRAPH

    • This button plots the graph of Debited or Credited (according to what is chosen from the option menu)

  • OTHER DATA

    • This button opens a window in which a database entry form is created which is used to add number of people who have worked on a specific day.The "add people type" button in this form is used to add the different category of people working.eg: plumber,painter,etc

  • RESET

    • This button is used to delete all the information in the tally database.Note: If the databse has not been uploaded then all the information will get lost.But this wont erase any information in "other data"

  • EXIT

    • This button is used to exit the application, it is not necessary to exit the application using this button.You also use the cross arrow at the top to exit and no data will get lost

  • THEME

    • This is an upcoming feature

  • RELOAD

    • This reloads the database window.This can be used to rectify any errors that occurs in the screen.

  • Export

    This converts and exports the databse into excel file format and is saved as "masterfile.xlsx"

  • View

    This shows all the entries made in the tally app till date in one window.
